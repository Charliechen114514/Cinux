# Hands-on 动手实践

OS 作业式引导教程。只给设计描述和验证命令，**绝不展示代码**——你需要自己动手实现。

## 内容结构

按编号顺序组织，每个编号对应一个内核功能模块，每篇聚焦一个功能点：

| 编号 | 模块 | 说明 |
|------|------|------|
| 000 | 环境搭建 | 工具链配置、QEMU 安装、项目骨架 |
| 001 | Real Mode 引导 | BIOS 中断、MBR 结构、磁盘读取 |
| 002 | GDT 与保护模式 | 段描述符、GDT 加载、模式切换 |
| 003 | Long Mode | 页表、CR3 配置、64 位模式入口 |
| 004 | 内核加载 | Multiboot2、ELF 解析、内核入口 |
| 005+ | 内核功能 | 中断、内存管理、驱动、文件系统…… |

## 适合谁

- 有一定 C/C++ 基础，想从零写一个 OS 的开发者
- 了解 x86 基本概念（寄存器、内存分段）即可开始
- 不需要操作系统课程基础——每篇会交代必要的背景知识

## 推荐阅读路径

按编号顺序阅读，从 [环境搭建（000-1）](000-env-toolchain-1.md) 开始。每篇末尾有「验证」步骤，通过即可进入下一篇。

如果遇到问题，可以对照同编号的 [Read-through](/read-through/) 篇章查看参考实现。
