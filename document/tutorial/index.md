# Tutorial 最终发布版

叙事体 + 代码精讲 + 外部引用，面向最终发布的教程文章。替代传统博客，每篇自成体系。

## 内容结构

按功能模块编号，覆盖从环境搭建到多终端的完整学习路径。每篇独立可读，无需严格按顺序：

| 编号 | 模块 | 核心话题 |
|------|------|----------|
| 000 | 环境搭建 | WSL2 + QEMU + 交叉编译工具链 |
| 001 | Real Mode | BIOS 中断、MBR 引导扇区、磁盘读取 |
| 002 | 保护模式 | GDT、段描述符、模式切换原理 |
| 003 | Long Mode | 4 级页表、PAE、64 位模式入口 |
| 004 | 内核加载 | Multiboot2 协议、ELF 加载 |
| 005–008 | 内核基础设施 | 入口点、物理内存、中断、大内核加载 |
| 009+ | 驱动与系统 | 串口驱动、VFS、进程、Shell…… |

## 适合谁

- 想要**系统阅读**一篇完整教程的读者
- 不想跟着做题、只想理解原理和设计思路的人
- 对 Intel SDM 引用和 x86 规范细节感兴趣的开发者
