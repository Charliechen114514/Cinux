# Read-through 代码详解

完整代码 walkthrough，逐行讲解实现细节。适合想深入理解每一行代码含义的读者。

## 内容结构

与 Hands-on 一一对应，同编号章节覆盖相同的内核模块。每篇包含完整的源代码和逐行注释：

| 编号 | 模块 | 代码亮点 |
|------|------|----------|
| 000 | 环境搭建 | Makefile 结构、链接脚本、交叉编译工具链配置 |
| 001 | Real Mode 引导 | `int 0x10` / `int 0x13` 调用、FAT12 BPB 解析 |
| 002 | GDT 与保护模式 | 段描述符构造、`lgdt` 指令、CR0 切换 |
| 003 | Long Mode | 4 级页表映射、`cr3` 加载、64 位 GDT 重建 |
| 004 | 内核加载 | Multiboot2 tag 遍历、ELF header 校验、BSS 清零 |
| 005+ | 内核功能 | 内联汇编、模板元编程、freestanding 环境 |

## 适合谁

- 想看到**完整参考实现**的开发者
- 正在跟做 Hands-on、需要对照检查自己代码的读者
- 对 C++ freestanding / 内核级 C++ 编程感兴趣的人

## 推荐阅读路径

两种用法：

- **跟做 Hands-on 时对照**：做到某个编号卡住时，直接翻同编号的 Read-through
- **独立阅读**：从 [环境搭建（000-1）](000-env-toolchain-1.md) 开始，按编号顺序阅读

如果想快速了解整体流程再看细节，可以先浏览同编号的 [Tutorial](/tutorial/)。
